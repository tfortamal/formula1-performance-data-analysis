# Formula 1 Performance Data Analysis

This project explores historical Formula 1 race data to analyze driver performance, constructor dominance, race reliability, and pit stop efficiency using Python.

The analysis processes multiple structured datasets and generates visual insights into key performance indicators in Formula 1.

---

## Project Overview

The project builds a lightweight data analysis pipeline that:

- Loads and processes multiple F1 datasets
- Cleans and merges relevant tables
- Performs exploratory analysis on driver and team performance
- Generates static visualizations for performance insights

The goal is to better understand patterns in race outcomes, driver success, and team strategies.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- OpenCV (for image export)

---

## Key Insights Explored

The analysis investigates several important aspects of Formula 1 racing:

### Driver Performance
- Top 10 drivers by total championship points
- Top 10 drivers by fastest laps
- Drivers with the highest DNF (Did Not Finish) counts

### Qualifying vs Race Success
- Relationship between **pole positions and race wins**
- Correlation between starting advantage and race outcomes

### Team Performance
- Top constructors by total points
- Comparison of team performance over time

### Pit Stop Efficiency
- Fastest constructors based on **average pit stop duration**

---

## Generated Visualizations

The script generates the following charts:

- `Top 10 Drivers by Total Points`
- `Pole Positions vs Race Wins`
- `Top 10 Constructors by Total Points`
- `Top 10 Drivers by Fastest Laps`
- `Top 10 Fastest Constructors by Avg Pit Stop Duration`
- `Top 10 Drivers by DNF Count`

All visualizations are exported as **high-resolution PNG images**.
